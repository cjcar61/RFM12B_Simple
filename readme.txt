Simple demo for transmittting four vales from RFM12B RF tranceiver to another. 

Tx has been designed to simulate emonTx 
Rx has been designed to simulate emonBase without the web-connection part. Rx sketch could be used to obtain wireless enegy monitoring data from an actual emonTx

Based on code from JCW at Jeelabs.org.  

JeeLib library from Jeelabs required: https://github.com/jcw/jeelib
