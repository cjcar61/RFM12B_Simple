Simple demo for transmittting two vales (one integer and one float) from RFM12B RF tranceiver to another. 

Based on code from JCW at Jeelabs.org.  

RFM12  and Ports library from Jeelabs required: http://jeelabs.net/projects/cafe/wiki

RFM12B documentation: http://openenergymonitor.org/emon/node/271
